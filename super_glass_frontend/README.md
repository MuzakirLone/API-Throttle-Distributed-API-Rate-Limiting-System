
# Super Glass Backend Dashboard

A ready-to-run React + Vite + Tailwind + Framer Motion frontend to exercise your backend endpoints with a modern glassmorphic UI.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173 and point Base URL to your backend (default http://localhost:3000).

## Features
- Glassmorphism: blurred translucent cards, soft borders, glow shadows
- Method selector (GET/POST/PUT/PATCH/DELETE)
- Headers editor (k:v per line)
- Request body editor (for non-GET)
- Quick endpoint shortcuts
- Burst x20 tester with average latency
- Status/latency badges
